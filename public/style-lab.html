Brand foundations

The system behind every Cabinet surface: themes, colors, type, components, and motifs. Full written spec lives in docs/brand-guide.md.

Theme

Warm, owned, human. A parchment-and-brown world that feels like a calm, well-made workspace, not a cold SaaS dashboard. Confident and plain-spoken, specificity over adjectives, one primary action per view. Sage green means "good / included." Audience: C-level and the technical leaders who vet the tool.

Liquid glass

The site chrome (nav, dropdowns, drawers) is liquid glass: frosted blur + saturation, a bright specular edge, and real refraction of the content beneath. The nav is floating glass buttons, with one solid accent button as the single primary action.

CSS backdrop-filter (blur + saturate) plus an SVG displacement filter for the refraction. Technique inspired by liquid-glass-js. Best viewed in Chrome.

Motifs

Illustrations are warm light wood (soft matte, wooden-toy craft, blonde/maple) with small touches of color: file-type cards and glowing agent tokens accent the meaningful parts. One iconic subject per page, on a transparent background. More in the galleries below.

Assets & re-run

Every generated asset on this page is a deterministic file = its id. To re-run one, name the id and we run its generator, for example node scripts/generate-images.mjs home-hero, node scripts/explore-styles.mjs cabinet (add STYLES=02,03 to limit), node scripts/brand-states.mjs, or the icon-sprite script.
